# Geometricshapes

Geometricshapes is a Java based programm that creates circles, rectangles
based on individual users input.

---

## Installation

Download an IDE on your Computer which should have a JavaKit 
then download and unpackage the [zip](C:\Users\PC\IdeaProjects\GeometricShapes.zip) file. Open the file in your IDE.


## Usage
Start the the GeometricShapes Klass afterwards in your terminal you should 
see.
```java
Available commands:
New rectangles - create new rectangles 
New circle - create new circle
Quit - quit program
```
Upper lower case writing is unimportant 
if user types new rectangle, user is asked to type
1. height
2. width
3. area

new circle offers the same questions. Afterwards user sees the rectangle or circle.
With quit the user can stop the programm.
