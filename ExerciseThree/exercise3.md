# Testfälle für den Calculator

## Addition
- Addition von zwei positiven Zahlen (5 + 3 = 8)
- Addition einer positiven und einer negativen Zahl (5 + (-3) = 2)
- Addition mit 0 als zweitem Operanden (5 + 0 = 5)

## Subtraktion
- Subtraktion zweier positiver Zahlen (5 - 3 = 2)
- Subtraktion einer negativen Zahl (5 - (-3) = 8)
- Subtraktion mit 0 als zweitem Operanden (5 - 0 = 5)

## Multiplikation
- Multiplikation zweier positiver Zahlen (5 * 3 = 15)
- Multiplikation mit einer negativen Zahl (5 * (-3) = -15)
- Multiplikation mit 0 (5 * 0 = 0)

## Division
- Division zweier positiver Zahlen (6 / 3 = 2)
- Division mit einer negativen Zahl (6 / (-3) = -2)
- Division durch 0: Es wird geprüft, dass eine ArithmeticException geworfen wird
